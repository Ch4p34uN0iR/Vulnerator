// Get Started

To begin, please select a download option to the left:

  • Release: The latest compiled version of the software - unless you are a coder or want to review the source for the application, this is the button for you.
  • .zip: A compressed folder containing the latest source for the application1
  • .tar.gz: A compressed folder containing the latest source for the application1

1If you are interested in viewing the source (or contributing), we recommend forking the repository on GitHub

// Using the Release

Once you have downloaded the latest release file, unzip the contents into a folder2. It is important that you extract all of the files, as there are hidden dependancies included with the visible executable ("*.exe") file.

2The release ZIP file contains a folder containing the executable and all dependencies - you can either extract this folder directly or extract its contents into a folder of your choice.

Once extracted, launch the application by double-clicking the "Vulnerator.exe" file within the folder it was extracted to. If a shortcut is desired, one can be created by right-clicking the "Vulnerator.exe" file and selecting "Send To", then "Desktop (Create Shortcut)".
